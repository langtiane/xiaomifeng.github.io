🔴🟢🔵 光的三原色大战

分数: 0 | 生命值: 3
炮弹
鼠标移动 = 瞄准
鼠标点击 = 发射炮弹
红+绿=黄 | 红+蓝=品红 | 绿+蓝=青 | 三色全=白

游戏规则

1. 游戏目标：使用光的三原色合成炮弹，消灭对应颜色的敌人

2. 操作方法：

- 鼠标移动：瞄准敌人

- 鼠标点击：发射炮弹

- 红、绿、蓝按钮：合成炮弹颜色

3. 颜色合成规则：

- 红 + 绿 = 黄

- 红 + 蓝 = 品红

- 绿 + 蓝 = 青

- 红 + 绿 + 蓝 = 白

4. 游戏机制：

- 每发射一次炮弹后需要重新合成

- 敌人触底会减少生命值

- 共有3条生命值

- 消灭敌人获得分数
